Welcome to skyloc's documentation!
==================================

**skyloc** is a Solar System object locator and ephemeris toolkit for
Python.  It provides classes and utilities for locating Solar System
objects in astronomical images, querying orbital databases (JPL SBDB /
Horizons), propagating orbits, and working with FITS header databases
and WCS transformations.
